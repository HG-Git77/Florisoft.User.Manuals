# Florisoft Hub app installation manual

This manual helps you download, install and set up the Florisoft Hub app. Through the Hub app, you can download, open, remove and later update other Florisoft apps.

## Before you start

Make sure you have the following information available:

- The Cloud server URL of your Florisoft environment.
- A Florisoft user account that is allowed to log in.
- An internet connection on the device on which you install the Hub app.

Choose the version that matches the device on which you want to use Florisoft apps. The available apps can differ between Android and Windows.

## Downloading the Android app

For Android devices, scan the QR code below.

<img src="Media/Installatie Hub-App/image1.png" width="500" height="500">

Unable to scan the QR code? Open this link in an internet browser on the Android device:

https://app.florisoft.nl/apps/latest/hub/android/com.florisoft.hub-signed.apk

## Installing the Android app

Perform these steps on the Android device on which you want to use the Florisoft apps. After the Hub app has been installed, first connect it to the main system. After that, you can download and open the available apps.

1. **Download and open the installation file**

   Scan the QR code with the Android device or open the download link in an internet browser.

2. **Allow the installation**

   If Android shows a message about installing from unknown sources, allow the installation for the browser or file app you used to open the file. Then go back and finish the installation. When the installation is successful, the Hub app will be listed with the other apps on the device.

3. **Open the Hub app and go to the settings**

   Once the download is complete, open the Hub app. Tap the menu icon in the top-right corner and then tap **Settings**.

   <details>
   <summary><b>Show example image</b></summary>

   <img src="Media/Installatie Hub-App/image12.png">
   <img src="Media/Installatie Hub-App/image13.png">

   </details>

4. **Connect the Hub app to the main system**

   Enter the following settings:

   - **Cloud server URL**: enter the URL of your Florisoft environment.
   - **Release channel**: usually enter `latest`. Use `update` only when Florisoft asks you to use an update or test version.
   - **Language**: select the display language of the app.

   Save the settings. The app asks to restart. Tap **Yes**.

   <details>
   <summary><b>Show example image</b></summary>

   <img src="Media/Installatie Hub-App/image14.png">
   <img src="Media/Installatie Hub-App/image15.png">

   </details>

5. **Open the catalog and log in**

   After the restart, tap **Catalog**. Then log in with your Florisoft user account. This can be a dedicated Hub app user or another Florisoft system user that is not already in use.

   <details>
   <summary><b>Show example image</b></summary>

   <img src="Media/Installatie Hub-App/image16.png">
   <img src="Media/Installatie Hub-App/image17.png">

   </details>

6. **Download an app**

   You now see the apps that are available for your Florisoft environment. Swipe an app to the left to show the buttons. Tap the download icon to download the app. Use the delete icon to remove an app.

   <details>
   <summary><b>Show example image</b></summary>

   <img src="Media/Installatie Hub-App/image19.png">

   </details>

7. **Open or find the downloaded app**

   Has the app been downloaded? Tap the arrow next to the app to open it from the Hub app. You can also open the app from the app list on the Android device. Search for the app name as shown in the Hub app.

The Hub app is now ready to use.

Important: do not remove the Hub app while you use other Florisoft apps. The Hub app is needed to manage and update these apps when new versions become available.

## Downloading the Windows app

For the Windows desktop app, use this download link:

https://app.florisoft.nl/apps/latest/hub/windows/hub.zip

## Installing the Windows app

Install the Windows Hub app when you want to use or manage Florisoft apps on Windows. The available apps can differ from the apps that are available on Android.

After opening the download link, a `.zip` file will be downloaded.

1. **Open the download folder**

   Go to the folder where the `.zip` file was downloaded.

2. **Create a fixed installation folder**

   Create a fixed folder for the Hub app. For example:

   `C:\FlorisoftApps\<CustomerName>\Hub\bin`

   Do not use spaces in the folder names.

   <details>
   <summary><b>Show example image</b></summary>

   <img src="Media/Installatie Hub-App/image2.png">

   </details>

3. **Extract the file**

   Extract the `.zip` file into the `bin` folder. The files must be placed in a folder named `bin`, because the update process expects this folder. Do not place the files in the downloads folder or in a temporary folder.

4. **Create a shortcut**

   In the `bin` folder, find the file **Florisoft.Hub.Desktop**. Create a shortcut for this file on the desktop.

5. **Open the Hub app**

   Open the Hub app through the shortcut. Click the menu icon in the top-right corner.

   <details>
   <summary><b>Show example image</b></summary>

   <img src="Media/Installatie Hub-App/image3.png">

   </details>

6. **Open the settings**

   Click **Settings**.

   <details>

   <img src="Media/Installatie Hub-App/image4.png">

   </details>

7. **Enter the settings**

   Enter the following settings:

   - **Cloud server URL**: enter the URL of your Florisoft environment.
   - **Release channel**: usually enter `latest`. Use `update` only when Florisoft asks you to use an update or test version.
   - **Language**: select the display language of the app.

   Save the settings afterwards.

## Frequently asked questions

### The QR code does not work

Open the Android download link manually in an internet browser on the Android device.

### Android blocks the installation

Allow installation from this source for the browser or file app you use to open the APK file. This is required because the Hub app is installed outside the Google Play Store.

### I do not know which release channel to use

Use `latest`, unless Florisoft has asked you to use `update`.

### Why is an app greyed out?

If an app is greyed out in the Hub app, it is not available for the platform on which you are using the Hub app. If you see this on Android, the app is only available for Windows. If you see this on Windows, the app is only available for Android.

### I do not see any apps in the catalog

Check the Cloud server URL, the release channel and the Florisoft user account you are using. Contact Florisoft if the settings are correct but the catalog remains empty.
