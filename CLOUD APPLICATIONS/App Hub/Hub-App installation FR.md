# Manuel d'installation de l'application Hub Florisoft

Ce manuel vous aide à télécharger, installer et configurer l'application Hub de Florisoft. Avec l'application Hub, vous pouvez télécharger, ouvrir, supprimer et mettre à jour d'autres applications Florisoft.

## Avant de commencer

Assurez-vous d'avoir les informations suivantes à portée de main :

- L'URL du serveur cloud de votre environnement Florisoft.
- Un utilisateur Florisoft autorisé à se connecter.
- Une connexion internet sur l'appareil sur lequel vous installez l'application Hub.

Choisissez la version qui correspond à l'appareil sur lequel vous souhaitez utiliser les applications Florisoft. Les applications disponibles peuvent différer entre Android et Windows.

Pour les appareils Android, scannez le QR code ci-dessous.

## Installer l'application Android

Effectuez ces étapes sur l'appareil Android sur lequel vous souhaitez utiliser les applications Florisoft. Après l'installation de l'application Hub, connectez-la d'abord au système principal. Vous pourrez ensuite télécharger et ouvrir les applications disponibles.

1. **Télécharger et ouvrir le fichier d'installation**

   Scannez le QR code avec l'appareil Android ou ouvrez le lien de téléchargement dans un navigateur internet.

2. **Autoriser l'installation**

   Si Android affiche un message concernant l'installation depuis des sources inconnues, autorisez l'installation pour le navigateur ou l'application de fichiers avec laquelle vous avez ouvert le fichier. Revenez ensuite en arrière et terminez l'installation. Lorsque l'installation est réussie, l'application Hub apparaît parmi les applications de l'appareil.

3. **Ouvrir l'application Hub et accéder aux paramètres**

   Une fois le téléchargement terminé, ouvrez l'application Hub. Appuyez sur l'icône de menu en haut à droite, puis sur **Settings**.

   <details>
   <summary><b>Afficher l'image d'exemple</b></summary>

   <img src="Media/Installatie Hub-App/image12.png">
   <img src="Media/Installatie Hub-App/image13.png">

   </details>

4. **Connecter l'application Hub au système principal**

   Renseignez les paramètres suivants :

   - **URL du serveur cloud** : saisissez l'URL de votre environnement Florisoft.
   - **Canal de publication** : saisissez généralement `latest`. Utilisez `update` uniquement lorsque Florisoft vous demande d'utiliser une version de mise à jour ou de test.
   - **Langue** : choisissez la langue d'affichage de l'application.

   Enregistrez les paramètres. L'application demande à redémarrer. Appuyez sur **Yes**.

   <details>
   <summary><b>Afficher l'image d'exemple</b></summary>

   <img src="Media/Installatie Hub-App/image14.png">
   <img src="Media/Installatie Hub-App/image15.png">

   </details>

5. **Ouvrir le catalogue et se connecter**

   Après le redémarrage, appuyez sur **Catalog**. Connectez-vous ensuite avec votre utilisateur Florisoft. Il peut s'agir d'un utilisateur dédié à l'application Hub ou d'un autre utilisateur système Florisoft qui n'est pas déjà utilisé.

   <details>
   <summary><b>Afficher l'image d'exemple</b></summary>

   <img src="Media/Installatie Hub-App/image16.png">
   <img src="Media/Installatie Hub-App/image17.png">

   </details>

6. **Télécharger une application**

   Vous voyez maintenant les applications disponibles pour votre environnement Florisoft. Balayez une application vers la gauche pour afficher les boutons. Appuyez sur l'icône de téléchargement pour télécharger l'application. Utilisez l'icône de suppression pour supprimer une application.

   <details>
   <summary><b>Afficher l'image d'exemple</b></summary>

   <img src="Media/Installatie Hub-App/image19.png">

   </details>

7. **Ouvrir ou retrouver l'application téléchargée**

   L'application est téléchargée ? Appuyez sur la flèche à côté de l'application pour l'ouvrir depuis l'application Hub. Vous pouvez également l'ouvrir depuis la liste des applications de l'appareil Android. Recherchez le nom de l'application tel qu'il est affiché dans l'application Hub.

L'application Hub est maintenant prête à être utilisée.

Important : ne supprimez pas l'application Hub tant que vous utilisez d'autres applications Florisoft. L'application Hub est nécessaire pour gérer ces applications et les mettre à jour lorsque de nouvelles versions sont disponibles.

## Télécharger l'application Windows

Pour l'application Windows, utilisez ce lien de téléchargement :

https://app.florisoft.nl/apps/latest/hub/windows/hub.zip

## Installer l'application Windows

Installez l'application Hub Windows lorsque vous souhaitez utiliser ou gérer des applications Florisoft sur Windows. Les applications disponibles peuvent différer de celles disponibles sur Android.

Après avoir ouvert le lien de téléchargement, un fichier `.zip` est téléchargé.

1. **Ouvrir le dossier de téléchargement**

   Accédez au dossier dans lequel le fichier `.zip` a été téléchargé.

2. **Créer un dossier d'installation fixe**

   Créez un dossier fixe pour l'application Hub. Par exemple :

   `C:\FlorisoftApps\<NomClient>\Hub\bin`

   N'utilisez pas d'espaces dans les noms de dossiers.

   <details>
   <summary><b>Afficher l'image d'exemple</b></summary>

   <img src="Media/Installatie Hub-App/image2.png">

   </details>

3. **Extraire le fichier**

   Extrayez le fichier `.zip` dans le dossier `bin`. Les fichiers doivent se trouver dans un dossier nommé `bin`, car le processus de mise à jour attend ce dossier. Ne placez pas les fichiers dans le dossier de téléchargement ou dans un dossier temporaire.

4. **Créer un raccourci**

   Dans le dossier `bin`, recherchez le fichier **Florisoft.Hub.Desktop**. Créez un raccourci vers ce fichier sur le bureau.

5. **Ouvrir l'application Hub**

   Ouvrez l'application Hub depuis le raccourci. Cliquez sur l'icône de menu en haut à droite.

## Questions fréquentes

### Le QR code ne fonctionne pas

Ouvrez manuellement le lien de téléchargement Android dans un navigateur internet sur l'appareil Android.

### Android bloque l'installation

Autorisez l'installation depuis cette source pour le navigateur ou l'application de fichiers avec laquelle vous ouvrez le fichier APK. Cela est nécessaire, car l'application Hub est installée en dehors du Google Play Store.

### Je ne sais pas quel canal de publication utiliser

Utilisez `latest`, sauf si Florisoft vous a demandé d'utiliser `update`.

### Pourquoi une application est-elle grisée ?

Si une application est grisée dans l'application Hub, elle n'est pas disponible pour la plateforme sur laquelle vous utilisez l'application Hub. Si vous voyez cela sur Android, l'application est uniquement disponible pour Windows. Si vous voyez cela sur Windows, l'application est uniquement disponible pour Android.

### Je ne vois aucune application dans le catalogue

Vérifiez l'URL du serveur cloud, le canal de publication et l'utilisateur Florisoft utilisé. Contactez Florisoft si les paramètres sont corrects mais que le catalogue reste vide.
