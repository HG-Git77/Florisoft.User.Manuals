# Installation de l'application Hub de Florisoft
Ce document décrit l'installation et la configuration initiale de l'application Hub de Florisoft. Vous y trouverez les étapes pour télécharger et installer l'application, ainsi que son fonctionnement en combinaison avec d'autres applications disponibles.

## Table des matières
<br>[Téléchargement de l'application pour Android & Windows](#téléchargement-de-lapplication-pour-android--windows)</br>
<br>[Installation de l'application Windows](#installation-de-lapplication-windows)</br>
<br>[Installation de l'application Android](#installation-de-lapplication-android)</br>

## Téléchargement de l'application pour Android & Windows
Commencez par télécharger l'application. Pour les appareils Android, scannez le QR code ci-dessous. Pour la version Windows, utilisez ce lien : https://app.florisoft.nl/apps/latest/hub/windows/hub.zip
<img src="Media/Installatie Hub-App/image1.png" width="500" height="500">

*Le lien peut également être saisi manuellement dans un navigateur : https://app.florisoft.nl/apps/latest/hub/android/com.florisoft.hub-signed.apk*

Si nécessaire, vous pouvez utiliser la version **Trusted** de l'APK.

<img src="Media/Installatie Hub-App/QR Code Trusted.png" width="500" height="500">

*Le lien peut également être saisi manuellement dans un navigateur : https://app.florisoft.nl/apps/latest/hub/android/com.florisoft.hub-trusted.apk*

## Installation de l'application Windows

L'installation de l'application Hub pour Windows est facultative ; en général, la version Android est utilisée. Vous n'avez donc pas besoin de la version Windows pour gérer vos applications.

Après avoir cliqué sur le lien, un fichier .zip sera téléchargé.

|**Étape**|Explication|
|-|-|
|1| Accédez au dossier où le fichier .zip a été téléchargé.|
|2| Créez un dossier, par exemple sur le disque C:\ nommé FlorisoftApps, avec un sous-dossier pour le client, un dossier Hub, et un dossier Bin. Exemple : C:\FlorisoftApps\(NomClient)\Hub\bin <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image2.png"></details>|
|3| Cliquez avec le bouton droit sur le fichier zip et extrayez les fichiers dans le dossier bin, par exemple : C:\FlorisoftApps\(NomClient)\Hub\bin\. Important : les fichiers doivent toujours être placés dans un dossier nommé bin. Le processus de mise à jour s’attend à cela. Veillez également à ne pas inclure d’espaces dans les noms des dossiers.|
|4| Dans le dossier, vous trouverez un fichier nommé 'Florisoft.Hub.Desktop'. Créez un raccourci vers ce fichier sur le bureau.|
|5| Ouvrez l'application. Vous verrez un écran. Cliquez sur l'icône hamburger en haut à droite. <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image3.png"></details>|
|6| Cliquez sur 'Settings'. <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image4.png"></details>|
|7| Saisissez l’URL du serveur cloud dans le premier champ.<br>Dans le deuxième champ, sélectionnez un canal de publication. Il existe deux options :<br><br>**1. latest** : correspond à la version utilisée avec votre installation principale.<br><br>**2. update** : permet de tester des versions plus récentes via le client de mise à jour.<br><br>Entrez "latest" ou "update". En général, on utilise "latest".<br>Dans le troisième champ, choisissez la langue d'affichage. Sauvegardez ensuite les modifications.<details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image5.png"></details>|
|8| L'application va redémarrer. Cliquez sur 'Oui'/'Yes'. <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image6.png"></details>|
|9| Cliquez ensuite sur le bouton 'Catalog'/'Catalogue'. <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image7.png"></details>|
|10| Connectez-vous avec votre utilisateur Florisoft (cela peut être un utilisateur dédié à la Hub-app ou tout autre utilisateur FS non encore connecté).<details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image8.png"></details>|
|11| Vous verrez un aperçu des applications disponibles. Pour en télécharger une, cliquez et maintenez avec le bouton gauche de la souris tout en la déplaçant vers la gauche. Deux icônes apparaîtront : télécharger (1) ou supprimer (2). <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image9.png"><img src="Media/Installatie Hub-App/image10.png"></details>|
|12| Une fois téléchargée, cliquez sur la flèche pour ouvrir l'application. <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image11.png"></details>|
|13| Vous trouverez également l'application dans l’explorateur de fichiers sous C:\Utilisateurs\VOTRENOM. Le dossier portera le même nom que l'application téléchargée. Vous pouvez créer un raccourci vers ce fichier et le placer sur le bureau. <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image23.png"></details>|

:warning: Notez qu'une fois connecté à l'app Hub avec un utilisateur, cet utilisateur ne peut pas être utilisé pour d'autres applications. Il est possible de forcer une déconnexion, mais il est recommandé de créer un utilisateur spécifique dédié à l'app Hub.

Il est essentiel de garder l'application Hub installée si vous utilisez d'autres applications Florisoft. Les mises à jour futures nécessiteront la présence de l'app Hub.

## Installation de l'application Android

Après avoir scanné le QR code ou saisi le lien, l'application a été téléchargée.

|**Étape**|Explication|
|-|-|
|1| Accédez à l'application téléchargée : 'Hub'. <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image22.png"></details>|
|2| Ouvrez l'application. Cliquez sur l'icône hamburger en haut à droite. <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image12.png"></details>|
|3| Cliquez sur 'Settings'. <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image13.png"></details>|
|4| Saisissez l’URL du serveur cloud dans le premier champ.<br>Dans le deuxième champ, entrez "latest" ou "update".<br><br>**1. latest** : pour la version principale de votre système backoffice.<br>**2. update** : pour tester des versions plus récentes via le client de mise à jour.<br><br>Dans le troisième champ, choisissez la langue d'affichage. Sauvegardez les modifications. <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image14.png"></details>|
|5| L'application va redémarrer. Cliquez sur 'Oui'/'Yes'. <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image15.png"></details>|
|6| Cliquez ensuite sur le bouton 'Catalog'/'Catalogue'. <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image16.png"></details>|
|7| Connectez-vous avec votre utilisateur Florisoft (utilisateur dédié ou tout autre utilisateur non encore connecté). <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image17.png"></details>|
|8| Une fois connecté, balayez vers la gauche en maintenant votre doigt sur une app pour faire apparaître les options : télécharger (1) ou supprimer (2). <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image18.png"><img src="Media/Installatie Hub-App/image19.png"></details>|
|9| Une fois téléchargée, cliquez sur la flèche pour ouvrir l'application. <details><summary>**Cliquez ici pour voir l'exemple d'image !**</summary><img src="Media/Installatie Hub-App/image20.png"></details>|
|10| Vous pouvez également retrouver les applications sur votre appareil en recherchant le nom indiqué dans l'application Hub.|

:warning: Comme mentionné précédemment, évitez de connecter un même utilisateur à plusieurs applications. Utilisez de préférence un utilisateur dédié à l’app Hub.

Une fois l'installation terminée, vous pouvez utiliser l'application Hub pour télécharger et lancer les autres applications.

Il est essentiel de conserver l'application Hub installée pour pouvoir gérer les futures mises à jour d'autres apps Florisoft.
